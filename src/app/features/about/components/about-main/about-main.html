De la chaudronnerie au développement web

Développeur Web certifié DWWM, motivé par la création d'applications web modernes et fonctionnelles

Issue de la chaudronnerie aéronautique où j'ai passé plusieurs années à former et déformer toutes sortes d'alliages, notamment le titane avec une exigence de précision absolue, j'ai (re)découvert une passion pour la création numérique. Cette expérience industrielle m'a forgé une rigueur technique et une capacité d'adaptation qui se révèlent aujourd'hui précieuses dans le développement web.

Ma transition vers le numérique s'est concrétisée par une formation intensive chez ALT-RH Consulting, où j'ai acquis les fondamentaux : HTML/CSS, JavaScript, Java et les méthodes agiles. Cette première étape m'a permis de valider mon appétence pour le code et de découvrir l'écosystème du développement moderne. J'ai ensuite approfondi mes compétences chez Studi pour obtenir mon titre de Développeur Web et Web Mobile, me spécialisant dans une stack fullstack complète : Angular, TypeScript côté front-end, NestJS pour les API, avec PostgreSQL et MongoDB pour la persistance des données, le tout orchestré avec Docker pour les déploiements. Aujourd'hui, je conçois des applications web performantes et accessibles, en appliquant la même exigence de qualité qui guidait mon travail dans l'aéronautique. Mon approche combine rigueur technique et sens du design pour créer des solutions robustes et élégantes, toujours centrées sur les besoins des utilisateurs.
